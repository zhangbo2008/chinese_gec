## 使用说明

1. 保存预训练模型在data文件夹下

│   ├── bert_config.json  
│   ├── config.json  
│   ├── pytorch_model.bin  
│   └── vocab.txt  
├── bert_corrector.py  
├── config.py  
├── logger.py  
├── predict_mask.py  
├── README.md

2. 运行`bert_corrector.py`可以进行纠错。
```
python3 bert_corrector.py
```
3. 评估
