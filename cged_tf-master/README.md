# CGED_TF

#### 整体思路

结合比赛本身提供的数据，可以将该问题建模为一个标注问题，例如命名实体识别等。NER的经典方案是BiLSTM+CRF。

#### 特征

输入特征为四个，分为三类:

(1) bigram特征，包括当前和前一时间步的bigram特征 

(2) pos特征: 特征提取工具使用哈工大的LTP，任何其他开源工具都可以，目的是做词性标注

(3) char特征
